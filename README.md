# Learning-git
Learning git from W3School.

## Introduction
<p>This repo is useful for beginners to intermediates programmers, I start it for myself to learn git and decided to share with you as a cheat sheet.</p>
<p>If there is a problem or confusion, you can check www.w3school.com for more example and details or ask myself.</p>

#### Point
<p>I don't share how to install git or creating remote account in Github, Gitlab or other remote websites.
But it's crucial to choose your name (id) and your gmail account for Git installation properly because Git use it as encryption keys to assign your work. However it can be changed after installation but not recommended</p>

<p>You can check what id and gmail set for your git after installing with commands.</p>
<p>First open Git Bash, write commands below</p>

```
git config --global user.name
git config --global user.email
```
<p>It will show your global configuration which set for Git.</p>
